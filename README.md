# Unity Portfolio -- Interactive 3D Experience

This repository contains the source code for my interactive
**Unity-based portfolio**, a small open-world space where visitors can
walk around, interact with elements, and discover my featured projects
in an immersive way.

The experience is built to showcase both my **Unity development skills**
and my **game programming workflow**, including AI behavior systems,
animations, scene composition, UI, and performance optimization.

The main character is implemented as a fully custom active ragdoll system developed from scratch.Building on those core principles, I created my own controller using real-time torque and force stabilization to maintain posture, balance, and locomotion.
This approach enables physics-driven movement with natural reactions to the environment, offering a more dynamic and responsive behavior than traditional animation-only controllers.
